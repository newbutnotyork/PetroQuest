🛢️ Нефтяной Экспресс (PetroQuest)

**Нефтяной Экспресс** — это интерактивное приложение, созданное для наглядного изучения процессов нефтегазовой отрасли. Приложение объединяет в себе динамичные 2D-анимации физических процессов и интерактивный квиз для закрепления материала, обернутые в современный интерфейс.

## ✨ Основные возможности

- **9 детализированных анимаций:** От зарождения нефти в порах породы до её переработки и транспортировки.
- **Интерактивный квиз:** Тестирование знаний с подробным разбором правильных ответов и отслеживанием прогресса.
- **Современный UI/UX:** Красивый интерфейс на базе PyQt5 с поддержкой светлой и темной тем.
- **Кибер-эстетика:** Фоновые анимации частиц, глитч-эффекты для текста, градиентные элементы оформления и кастомный дизайн окна.
- **Готовность к релизу:** Встроенная утилита `resource_path` для корректной сборки проекта в `.exe` через PyInstaller.

## 🎬 Визуализированные процессы

Приложение включает анимации следующих этапов:
1. **Пористость** — строение порового пространства коллектора.
2. **Бурение** — процесс бурения скважины и работы оборудования.
3. **Перфорация** — создание каналов в обсадной колонне для притока нефти/газа.
4. **Движение (Режимы)** — фонтанный, газовый и водонапорный режимы добычи.
5. **ШГН** — работа станка-качалки (штангового насоса).
6. **Парафины** — очистка скважины от асфальтосмолопарафиновых отложений (АСПО).
7. **Ректификационная колонна** — процесс ректификации нефти на фракции в колонне.
8. **Транспорт** — трубопроводный, железнодорожный, водный и авто транспорт.
9. **АЗС** — финальный этап: заправка автомобиля топливом.

## 🛠 Технологический стек

- **Python**
- **PyQt5** — создание графического интерфейса, виджетов, кастомных эффектов.
- **PyGame** — рендеринг и отрисовка сложных физических анимаций.
- **NumPy / SciPy** — математические расчеты и интерполяция путей.
- **JSON** — хранение вопросов для квиза.

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/famousrose/PetroQuest.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd petroquest
   ```
3. Установите необходимые зависимости:
   ```bash
   pip install pygame pyqt5 numpy scipy
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

## 📂 Структура проекта

- `main.py` — Точка входа в приложение.
- `ui.py` — Логика главного окна, кастомные виджеты интерфейса.
- `animations.py` — Встроенный PyGame-виджет и все 2D-анимации нефтедобычи.
- `quiz.py` — Логика тестирования, подгрузка вопросов из JSON, расчет результатов.
- `utils.py` — Утилиты для работы с путями.
- `questions.json` — База данных вопросов для квиза.
